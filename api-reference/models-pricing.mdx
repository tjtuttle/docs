<Note>
Pricing is per **1 million tokens** (1M tokens). You only pay for what you use.
</Note>

## Switching Between Models

You can easily switch between models by changing the `model` parameter:

<CodeGroup>

```bash Free Tier Model
curl -X POST https://api.ctgt.ai/v1/chat/completions \
  -H "Authorization: Bearer sk-ctgt-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

```bash Paid Tier Model
curl -X POST https://api.ctgt.ai/v1/chat/completions \
  -H "Authorization: Bearer sk-ctgt-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-sonnet-4-5-20250929",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

</CodeGroup>

## Model Availability Check

You can check which models are available for your tier:

```bash
curl https://api.ctgt.ai/v1/subscription/info \
  -H "Authorization: Bearer sk-ctgt-YOUR_API_KEY"
```

**Response includes:**
```json
{
  "subscription_tier": "free",
  "available_models": [
    {
      "model": "gemini-2.5-flash",
      "display_name": "Gemini 2.5 Flash",
      "tier": "free",
      "pricing": {
        "input": 0.50,
        "output": 2.70
      }
    }
  ]
}
```
