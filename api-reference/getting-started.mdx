## Quick Start

Get started with the CTGT API in 3 simple steps:

1. **Sign up** and verify your email
2. **Get your API key**
3. **Make your first request**

---

## Step 1: Create Your Account

Create your account with email verification:

<CodeGroup>

```bash cURL
curl -X POST https://api.ctgt.ai/v1/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "organization": "Acme Corp"
  }'
```

```python Python
import requests

url = "https://api.ctgt.ai/v1/auth/signup"
payload = {
    "name": "John Doe",
    "email": "john@example.com",
    "organization": "Acme Corp"
}

response = requests.post(url, json=payload)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.ctgt.ai/v1/auth/signup', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'John Doe',
    email: 'john@example.com',
    organization: 'Acme Corp'
  })
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

**Response:**
```json
{
  "message": "Account created successfully. Please check your email to verify your account.",
  "email": "john@example.com"
}
```

## Step 2: Verify Your Email

Check your inbox for a verification email containing a link like:

```
https://api.ctgt.ai/v1/auth/verify?token=VERIFICATION_TOKEN
```

Click the link or make a GET request:

<CodeGroup>

```bash cURL
curl "https://api.ctgt.ai/v1/auth/verify?token=VERIFICATION_TOKEN"
```

```python Python
import requests

token = "VERIFICATION_TOKEN"
url = f"https://api.ctgt.ai/v1/auth/verify?token={token}"

response = requests.get(url)
data = response.json()
print(f"API Key: {data['api_key']}")
```

</CodeGroup>

**Response:**
```json
{
  "message": "Email verified successfully!",
  "email": "john@example.com",
  "api_key": "sk-ctgt-abc123def456..."
}
```

<Warning>
**Save your API key!** You'll need it for all API requests. Store it securely and never share it publicly.
</Warning>

## Step 3: Make Your First Request

Now you can start making API requests using your API key:

<CodeGroup>

```bash cURL
curl -X POST https://api.ctgt.ai/v1/chat/completions \
  -H "Authorization: Bearer sk-ctgt-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash",
    "messages": [
      {"role": "user", "content": "Hello! What can you help me with?"}
    ]
  }'
```

```python Python
import requests

api_key = "sk-ctgt-YOUR_API_KEY"
url = "https://api.ctgt.ai/v1/chat/completions"

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

data = {
    "model": "gemini-2.5-flash",
    "messages": [
        {"role": "user", "content": "Hello! What can you help me with?"}
    ]
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result['choices'][0]['message']['content'])
```

```javascript JavaScript
const response = await fetch('https://api.ctgt.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer sk-ctgt-YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gemini-2.5-flash',
    messages: [
      { role: 'user', content: 'Hello! What can you help me with?' }
    ]
  })
});

const data = await response.json();
console.log(data.choices[0].message.content);
```

</CodeGroup>

**Response:**

## Authentication

All API requests require authentication using a Bearer token:

```
Authorization: Bearer YOUR_API_KEY
```

Your API key:
- Starts with `sk-ctgt-`
- Is unique to your account
- Should be kept secure and never shared
- Can be regenerated if compromised

<Tip>
Store your API key as an environment variable:
```bash
export CTGT_API_KEY="sk-ctgt-YOUR_API_KEY"
```
</Tip>
